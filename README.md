
 AI Stock Price Predictor

A Machine Learning web app built using Streamlit that predicts stock prices using historical or live data.

---

 Features

-  Display stock data  
- Interactive chart  
-  Live stock data (Yahoo Finance)  
-  Machine Learning prediction  
-  Predict Close price  

---

##  ML Model

- Algorithm: Linear Regression  
- Input: Open, High, Low  
- Output: Close price  

---

##  Tech Stack

- Python  
- Streamlit  
- Pandas  
- NumPy  
- Scikit-learn  
- yfinance  

---

##  Project Structure

app.py data.csv requirements.txt README.md

---

##  How to Run

### 1. Install dependencies
pip install -r requirements.txt

### 2. Run project
streamlit run app.py

---

 Usage

- Select CSV or Live Data  
- Choose stock  
- View chart  
- Enter values  
- Click Predict
